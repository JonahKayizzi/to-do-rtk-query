# ğŸ“– [TODO List with RTK & JSON Server] <a name="about-project"></a>

> This is a basic TODO List App with basic CRUD operations

Users can add new todo items, update, delete and view all todos.
The Todo items are stored and retrieved from a JSON file that is accessed using the json-server library at a defined port
