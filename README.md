# 📖 [TODO List with RTK & JSON Server] <a name="about-project"></a>

> This is a basic TODO List App with basic CRUD operations

Users can add new todo items, update, delete and view all todos.
The Todo items are stored and retrieved from a JSON file that is accessed using the json-server library at a defined port

## 🛠 Built With <a name="built-with">Javascript, React, Redux, RTK Query, npm, Gitflow, JSON Server</a>

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

run the following commands in your terminal

### Prerequisites

In order to run this project you need:

You need to have node installed

### Setup

Clone this repository to your desired folder:

- git clone https://github.com/JonahKayizzi/to-do-rtk-query.git
- cd to-do-rtk-query
- git checkout dev

### Install

Install this project with:

npm install

### Usage

To run the project, execute the following command:

run npm start

### Run tests

To run tests, run the following command:

run npm test

### Deployment

You can deploy this project using:

run npm build
